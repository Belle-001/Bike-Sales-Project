### Project Overview

This project focuses on analyzing bike sales data using Microsoft Excel to uncover customer purchasing patterns and key business insights. The goal was to transform raw sales data into a structured, interactive dashboard that supports data-driven decision-making.

### Objectives

- Clean and prepare the bike sales dataset for analysis
- Analyze customer demographics and purchasing behavior
- Identify key factors influencing bike purchases
- Build an interactive dashboard for easy insight exploration

### Dataset Description

**Source**: •	Bike_Sales_Excel_Project.xlsx - Public dataset 

**Content**: Customer demographics and purchase information

**Key fields**: 
- Age
- Gender
- Income
- Occupation
- Marital status
- Region
- Bike purchase(Yes/No)

### Results
Customers with higher average income tend to purchase more bikes.

The middle age group contributes the largest share of bike purchases.

Married males show the highest average income per purchase, indicating strong purchasing power.

Purchase declines as commute distance increases, indicating bikes are mainly used for short distance.  

### Recommendations
Invest in marketing and promotion to  married middle age people with shorter commute distance.
